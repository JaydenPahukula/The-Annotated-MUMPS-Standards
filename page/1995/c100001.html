American National Standard for Information Systems –
Computer Graphics –
GKS - MUMPS Language Binding

0 Introduction

The Graphical Kernel System (GKS) is registered as ANSI X3.124–1985. As explained in the Scope and Field of Applications of X3.124, that American National Standard is specified in a language independent manner and needs to be embedded in language dependent layers (language bindings) for use with particular programming languages.

The purpose of this document is to define a standard binding for the MUMPS programming language, ANSI X11.1–1990.
