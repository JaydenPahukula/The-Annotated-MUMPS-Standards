4.1.13 LINKSOURCE dattribute

Definition

LINKSOURCE provides, via descendant nodes, information about the data exchange sources available for the display. Each source is identified by an entry of the form

^$DI[SPLAY] ( display , linksourcekeyword , desource )

where

    linksourcekeyword::= expr V LINKSOURCE

and where desource subscripts indicate available data exchange sources.

No value is present at the level of the LINKSOURCE dattribute itself. For all descendant nodes, the value is an emptystring.

Domain emptystring
Access Reference
Default Not applicable
