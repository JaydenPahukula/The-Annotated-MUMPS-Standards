7.2.5 DMODE Draw Command

    drawmode::= expr V DMODE , dmode

The drawing mode determines the effect of certain drawcommands on pixels in the drawing area. If the value of dmode is equal to "SOLID", drawcommands cause pixels to be changed to the current draw foreground color or draw background color. If the value of dmode is equal to "XOR", the new color of each pixel is determined by performing a Boolean exclusive or (XOR) between the pixel in the drawing area and the current draw foreground color or draw background color.

The drawmode drawcommand specifies the current drawing mode. The specified mode is used by subsequent drawcommands until another drawmode drawcommand is specified. Prior to the first drawmode drawcommand for a drawing, the current drawing mode is SOLID.
