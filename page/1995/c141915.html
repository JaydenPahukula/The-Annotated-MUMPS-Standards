4.19.15 GQPPAR, Inquire predefined pattern representation

control-
mnemonic		   Level   GKS function name
GQPPAR 0a Inquire predefined pattern representation
2 parameters
p1 ::= gksnam workstation type
p2 ::= gksint predefined pattern index (1..n)
Write /GQPPAR("WORK1",1)
3 values returned
v1 ::= gksint error indicator
v2 ::= L gksint 2 value array dimensions (n,m) (1..n)
v3 ::= M gksint n*m pattern array (0..n)
Read Error,DimXY F ii=1:1:DimXY*$P(DimXY,",",2) R Pattern(ii)
