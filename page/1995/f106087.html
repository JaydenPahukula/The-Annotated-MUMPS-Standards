6.5.7 SEPARATOR cattribute

Definition

In a pop-up menu, the SEPARATOR cattribute causes a visual separator (typically a single horizontal line) to appear below the choice text of the choice for which it is specified.

In a menu bar, the first instance of the SEPARATOR cattribute causes subsequent menu choices to appear right justified within the menu bar, if appropriate for the underlying windowing platform. Any instances of the SEPARATOR cattribute after the first within a menu bar are ignored.

The only value that can be assigned to the SEPARATOR cattribute is an emptystring.

Domain emptystring
Access Create, Reference
Default None
