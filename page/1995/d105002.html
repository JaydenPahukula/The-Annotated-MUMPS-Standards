5.2 XTOOLKIT Specific Syntax

All of the routines in this package begin with the letters Xt followed by a mixed case string of characters, the first of which is capitalized, representing the XTOOLKIT routine to be executed. The routinenames used in this binding are those provided in the X Window System as proevided by the X Window Consortium. The exact list is not included herein. Those externrefs specified at this level that do not begin with the characters Xt are not included in this binding. Those entities defined in the standard XToolkit that are defined as macros beginning with the characters Xt also are not included.
