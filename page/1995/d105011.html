5.5 XMUMPS Specific Syntax

&XMUMPS.SetString(xmember,expr)

Modifies the value of the element of the xtype referenced by xpointer where xmember is a reference to the xtype. It is erroneous to use an xmember for which no xname exists. This notifies the underlying implementation that conversion to string type is needed. Actual type conversion is an implementation detail.
