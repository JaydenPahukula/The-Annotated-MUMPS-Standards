8.2.8 Hang

H[ANG] postcond SP L hangargument

    hangargument::= │
│
│		 numexpr
@ expratom V L hangargument		 │
│
│

Let t be the value of numexpr. If t '> 0, Hang has no effect. Otherwise, execution is suspended for t seconds.
