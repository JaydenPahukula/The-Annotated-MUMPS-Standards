5.4.24 Close client’s devices

(Operation type 43) relinquishes ownership of all devices owned by the client. This operation implies that the server node’s device manager shall retain the identifiers and nodes of clients owning devices (see B.5).

The request shall be 1 field:

  1. Client identifier: <SS> $Job (see ANSI/MDC X11.1) of the client process making the request, an ASCII string of decimal digits.
    NOTE – This operation is included for the convenience of implementers – there is no requirement that the request be sent in any specific condition.

The response shall consist of a header only. If output to any of the client’s devices has not finished, the server shall return the output pending error condition and shall not relinquish that device. Whether to relinquish other devices whose output has finished is left to the implementer.
