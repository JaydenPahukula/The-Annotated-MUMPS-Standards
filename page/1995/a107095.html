7.1.5.9 $Length

$L[ENGTH] ( expr )

This form returns an integer which is the number of characters in the value of expr. If the value of expr is the empty string, $Length(expr) returns the value 0.

$L[ENGTH] ( expr1 , expr2 )

This form returns the number plus one of nonoverlapping occurrences of expr2 in expr1. If the value of expr2 is the empty string, then $Length returns the value 0.
