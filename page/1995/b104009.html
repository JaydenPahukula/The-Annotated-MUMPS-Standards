4.4.2 Device

The agent shall send data in string write arguments to a device using the device’s character set profile. The agent shall refer to the server’s ssvns if necessary for the definition of the device’s character set profile. All other strings in device-related transactions shall be governed by the character translation field of the connect transaction as defined in 5.4.1.
