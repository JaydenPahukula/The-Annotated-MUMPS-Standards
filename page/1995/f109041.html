8.5.10 PSTATE einfoattribute

Definition PSTATE identifies any pointer buttons that were pressed when a pointer event occurred. If no buttons were pressed, PSTATE is equal to emptystring.
Domain L pbuttoncode
Access Reference
Default Not Applicable.
