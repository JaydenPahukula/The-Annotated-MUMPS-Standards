7.1.4.10 Intrinsic special variable names svn

Syntax Definition
$K[EY]

$Key contains the control-sequence which terminated the last Read command from the current device (including any introducing and terminating characters). If no Read command was issued to the current device or when no terminator was used, the value of $Key will be the empty string. The effect of a Read *glvn on $Key is unspecified.

If a Character Set Profile input-transform is in effect, then this is also applied to the value stored in $Key.

See (Read command) and (Write command).
