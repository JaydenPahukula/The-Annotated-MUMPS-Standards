6.4.28 LINKITEM elattribute

Definition Specifies the item of interest in the Data Exchange source identified in the element’s LINKSOURCE elattribute. For a TEXT gadget, data supplied by this item will be truncated at the first linend, if any.
Domain deitem
Access Create, Modify, Reference
Default emptystring
