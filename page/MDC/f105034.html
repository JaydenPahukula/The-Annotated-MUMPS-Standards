5.4.28 PTYPE wattribute

Definition

PTYPE specifies the pointer appearance when the pointer is in the window viewport.

If the display on which the window appears does not have a pointer, any value of PTYPE assigned by an application is ignored.

Domain ptype
Access Create, Modify, Reference
Default None
