3.3.18 Scrolling

scrollrange specifies the value range that a scroll or long list box gadget displays.

    scrollrange::= expr V │
│
│		 scrollmin
[ scrollmin ] , scrollmax		 │
│
│
    scrollmin::= numlit
    scrollmax::= numlit

scrollby specifies the amount by which the VALUE elattribute increases or decreases when the user presses a scroll gadget’s movement controls.

    scrollby::= expr V │
│
│		 scrollsmall
[ scrollsmall ] , scrollbig		 │
│
│
    scrollsmall::= numlit
    scrollbig::= numlit
