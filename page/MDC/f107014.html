7.2.11 FSTYLE Draw Command

    drawfstyle::= expr V FSTYLE , fontstyle

The drawfstyle drawcommand specifies the current font style. The specified font style is used by subsequent drawcommands until another drawfstyle drawcommand is specified. Prior to the first drawfstyle drawcommand for a drawing, the current font size is specified by the gadget’s FSTYLE elattribute. Requesting of a new font from the underlying windowing platform (if any) is deferred until the first subsequent drawcommand requiring a font.
