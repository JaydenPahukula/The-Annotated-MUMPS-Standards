6.1.6 List Box

A list box displays a list of items and enables the user to select one or more of them. If the list box is not large enough to show all items at once, a scrolling mechanism enables the user to move through the list. Such navigation through the list is handled automatically by the underlying windowing platform and the MWAPI. A list box has a TYPE elattribute value equal to "LIST".
