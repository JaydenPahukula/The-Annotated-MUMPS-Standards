9.5.4 ELEMENT einfoattribute

Definition If the event is associated with an element, ELEMENT specifies that element’s type and ename. If the event is not associated with an element, ELEMENT is not defined.
Domain For a gadget: G,ename
For a menu: M,ename
For a timer: T,ename
Access Reference
Default Not Applicable.
