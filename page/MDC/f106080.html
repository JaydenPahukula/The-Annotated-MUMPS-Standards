6.5.1 Applicability of Choice Attributes

The applicability of cattributes to element types is specified by Table 3. A "Yes" entry indicates that the use of the attribute with the element type is defined. An empty entry indicates that the use of the attribute with the element type is reserved. implementationattribute and applicationattribute values are applicable to all element types.
