9.2.2 FILTERIN especattribute

Definition Definition FILTERIN specifies, for certain events, additional criteria that must be TRUE for an event to occur.
Domain For a key down or key up event: L keycode
For a character event: L charcode
For a pointer event: L pbuttoncode
Access Create, Modify, Reference
Default None
