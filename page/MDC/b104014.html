4.5.4 Identification

A server shall be identified by 1 or more names that are available to agents. Likewise, an agent shall be identified by 1 or more names that are available to servers. The server and agent shall exchange their names when establishing a session.

Each request shall identify the user. Together, the agent’s name and the user’s identification furnish proof of origin, that is, the server "knows" the origin of a request. Completion of the transaction provides the agent with proof of delivery of the request.
