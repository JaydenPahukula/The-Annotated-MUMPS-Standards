4.1.17 PTYPE dattribute

Definition

If PTYPE is defined, it specifies the pointer appearance when the pointer is over the viewport of any window defined in ^$WINDOW which appears on the display. The PTYPE dattribute takes precedence over all PTYPE wattributes and over all PTYPE elattributes.

If the PTYPE dattribute is not defined and the pointer is not over a window or gadget for which a PTYPE wattribute or elattribute, respectively, is defined, the pointer appearance is determined by the platform.

Domain ptype
Access Modify, Reference
Default Not applicable
