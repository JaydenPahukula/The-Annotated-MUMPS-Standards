Sponsor

Not Yet Approved

Abstract

This standard contains a three-section description of various aspects of the M[UMPS] computer programming language. Section 1, the M[UMPS] Language Specification, consists of a stylized English narrative definition of the M[UMPS] language. Section 2, the M[UMPS] Portability Requirements, identifies constraints on the implementation and use of the language for the benefit of parties interested in achieving M[UMPS] application code portability. Section 3 is a binding to ANSI X3.64 (Terminal Device Control Mnemonics).
