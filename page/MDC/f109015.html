9.2.1 ENABLE especattribute

Definition An event identified by etype is enabled for a window, gadget, timer, or menu choice if, when the event occurs, there is an espec defined for the window, gadget, timer or menu choice which has a valid doargument for its value and the corresponding ENABLE especattribute for the espec has a TRUE value. If the ENABLE especattribute is FALSE, the event is not enabled for the window, gadget, timer or menu choice.
Domain truefalse
Access Create, Modify, Reference
Default TRUE
