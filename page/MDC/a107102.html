7.1.5.17 $QLength

$QL[ENGTH] ( namevalue )

See 7.1.4.12 for the definition of namevalue.

This form returns a value which is derived from namevalue. If namevalue has the form NAME(s1, s2, ..., sn), considering n to be zero if there are no subscripts, then the function returns n.

Note that the namevalue is not "executed", and will not affect the naked indicator, nor generate an error if the namevalue represents an undefined glvn. The naked indicator will only be affected by the last gvn reference (if any) executed while evaluating the parameter.
