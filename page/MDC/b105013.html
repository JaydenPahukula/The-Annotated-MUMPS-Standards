5.3.9.1 String

A string of characters (derived from WRITE expr) is sent to the device. In the record form of data the string shall be the value of 1 expr, and in the stream forms of data the string may also be part of 1 or more exprs (see 5.3.8).

  1. <SI> = 1
  2. <LS> the value of expr (from WRITE expr) is sent to the device.

The agent shall send this field in the character set profile of the device.
