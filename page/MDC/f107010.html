7.2.7 FCOLOR Draw Command

    drawfcolor::= expr V FCOLOR , color

The drawfcolor drawcommand specifies the current drawing foreground color. The specified color is used by subsequent drawcommands until another drawfcolor drawcommand is specified. Prior to the first drawfcolor drawcommand for a drawing, the current draw foreground color is determined by the gadget’s FCOLOR elattribute.
