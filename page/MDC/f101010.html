1.10 Build Mode

To support window layout tools windows may be created in build mode. The behavior of a window and its elements is modified in this mode. For example gadgets may be moved and resized but do not respond to normal user input, and timers do not run. A different set of events apply in build mode, all originated by the window. Selection, movement and resizing of one or more gadgets is supported in a platform-specific fashion by means of the pointer and/or keyboard.
