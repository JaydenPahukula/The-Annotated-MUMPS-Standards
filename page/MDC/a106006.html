6.2.3 Label label

Each occurrence of a label to the left of ls in a line is called a defining occurrence of label. An error condition occurs with ecode = "M57" if there are two or more defining occurrences of label with the same spelling in one routinebody.

Editor’s note:
The standard does not yet define the exact event that would trigger an error with ecode="M21". This error should most likely be triggered bt the execution of a RSave command.
