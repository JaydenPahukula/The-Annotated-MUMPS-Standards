5.4.32 SIZEMIN wattribute

Definition

SIZEMIN specifies the minimum size to which the user can shrink the window viewport, expressed in the unit of measure defined by the unitspec component of the size value, if present, and by the window’s UNITS wattribute otherwise. Any attempt by the user to make the window viewport smaller than SIZEMIN is ignored.

If the unit of measure is CHAR, the basis font is the font specified by the window’s FFACE, FSIZE and FSTYLE wattributes.

When an application assigns a value to the SIZEMIN wattribute of an existing window, if hsize or vsize is not specified then the minimum horizontal or vertical size of the window, respectively, is not altered.
