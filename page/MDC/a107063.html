7.1.4.10.6 $IO

$I[O]

identifies the current I/O device (see 8.2.7 and 8.2.35). Its value has the form of expr. When a process is initiated, but before any commands are processed, the value of $IO is equal to the value of $Principal if an implicit Open and Use for the device specified by $Principal is executed by the implementation. If the implementation does not execute these Open and Use commands then $IO is given the value of the empty string.
