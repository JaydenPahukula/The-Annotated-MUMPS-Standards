9.4.2 ESTOP

ESTO [ P ] postcond [ SP ]

The M[UMPS] command ESTOP implicitly performs the number of M[UMPS] QUIT commands necessary to return to the execution level of the most recently executed ESTART command and then terminates that ESTART command. If event processing is not activated, execution of an ESTOP command has no effect.
