7.2.1.1 Concatenation operator

The underscore symbol _ is the concatenation operator. It does not imply any numeric interpretation. The value of A_B is the string obtained by concatenating the values of A and B, with A on the left.

Editor’s note:
"... on the left" depends on the direction in which a text is being produced. It would be better to re-phrase this as "... with the value of A preceding the value of B."
($Extract(x,1) is the leftmost character when printing occurs "left-to-right", but the rightmost character when printing occurs "right-to-left".)
