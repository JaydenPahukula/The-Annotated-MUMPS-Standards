3.2 MUMPS controlmnemonics and GKS functions

Each GKS function is listed in the following section, grouped together by function type. Each function has an associated controlmnemonic which is the means by which it is invoked, a level, which indicates which versions of GKS will support the function, and the GKS function name which is the standard GKS description for the function.

See the index for the list of GKS functions sorted by function name or by controlmnemonic.

Each GKS function is presented with an example of how to use it in MUMPS, these lines begin with ‘’. In addition, GKS functions which return values include an example on receiving those values, these lines begin with ‘’.
