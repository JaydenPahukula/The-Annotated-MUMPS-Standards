6.5.5 EVENT cattribute

Definition EVENT specifies, via descendant nodes identified by etype subscripts, the choice’s callback routine(s) for the event specified by etype. The format of a choice event specification is defined in Section 9.2. No value is present at the level of the EVENT cattribute itself.
Domain For immediate descendants: L doargument
Access For immediate descendants: Create, Modify, Reference
Default None
