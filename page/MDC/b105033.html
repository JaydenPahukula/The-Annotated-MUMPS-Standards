5.4.9 Define

(Operation type 21) The define operation indicates whether the global variable given in its global reference field <LS> has a value and whether it has descendants.

<LS>
<LI> Global Reference

Figure 21 – Define request

The 1 field of the define response shall be a short integer <SI>, the $DATA value of the global reference. ANSI/MDC X11.1, 7.1.5.3, defines the $DATA values that indicate the presence of a global value and descendants.
