5.4.39 UNITS wattribute

Definition UNITS indicates the unit of measure used to specify the size and position of the window.
Domain unitspec
Access Create, Modify, Reference
Default For an application window: Value of parent’s UNITS dattribute or wattribute
For an MTERM window: "CHAR", with a basis font defined by an FFACE value of "M,FIXED", an FSIZE value of "12", and an FSTYLE value of "NORMAL".
