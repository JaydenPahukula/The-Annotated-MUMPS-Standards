5.4.1 Applicability of Window Attributes

The applicability of wattributes to window types is specified by Table 1. A "Yes" entry indicates that the use of the attribute with the window type is defined. An empty entry indicates that the use of the attribute with the window type is reserved. implementationattribute and applicationattribute values are applicable to all window types.
