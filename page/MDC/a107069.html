7.1.4.10.12 $Quit

$Q[UIT]

returns 1 if the current PROCESS-STACK frame was invoked by an exfunc or exvar, and therefore a Quit would require an argument. Otherwise, $Quit returns 0 (zero). When a process is initiated, but before any commands are processed, the value of $Quit is 0 (zero).
