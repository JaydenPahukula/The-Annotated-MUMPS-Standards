5.5 XMUMPS Specific Syntax

&XMUMPS.UnregisterEventHandler(xpointer)

Disables disables translation (if any) for labelref associated with the xpointer used as the fourth argument of &XTOOLKIT.ItAddEventHandler and other related Event Handler calls, such as &XTOOLKIT.AddRawEventHandler and &XTOOLKIT.RetioveEventHandler routines. This routine frees any memory allocated by the associated &XMUMPS.RegisterEventHandler call for use by the system.
