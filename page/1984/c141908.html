4.19.8 GQPMF, Inquire polymarker facilities

control-
mnemonic		   Level   GKS function name
GQPMF ma Inquire polymarker facilities
1 parameter
p1 ::= gksnam workstation type
Write /GQPMF("WORK1")
8 values returned
v1 ::= gksint error indicator
v2 ::= gksint number of available marker types (5..n)
v3 ::= M gksint n available marker types (-n..–1,1..n)
v4 ::= gksint number of available marker sizes (0..n)
v5 ::= gksnum nominal marker size (DC)
v6 ::= gksnum minimum marker size (DC)
v7 ::= gksnum maximum maximum size (DC)
v8 ::= gksint number of predefined polymarker indices (0,5..n)
Read Error,Num F ii=1:1:Num R MarkType(ii)
Read SizNum,SizNom,SizMin,SizMax,Markindx
