$PIECE function

See $Piece for formal definition.

  1. Result A.
  2. Result B.
  3. One ("1") C.
  4. Result C.
  5. C D.
  6. Result D.
  7. Place the C through D fields in A, delimited by B, into Result.

Note: The details of action 6 are described in Section 3.2.8 of Part I.
