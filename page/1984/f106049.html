6.4.29 LINKMODE elattribute

Definition

The value of LINKMODE controls the mode of operation of the element’s Data Exchange link, as defined by the elattributes LINKSOURCE and LINKITEM.

NONE No link is open
AUTOMATIC The element’s VALUE elattribute is updated whenever the item identified by LINKSOURCE and LINKITEM changes.
MANUAL The element is not informed of changes in the value of the item identified by LINKSOURCE and LINKITEM
NOTIFY The element is informed whenever the item identified by LINKSOURCE and LINKITEM changes, but the element’s VALUE elattribute is not updated.
Domain demode
Access Create, Modify, Reference
Default emptystring
