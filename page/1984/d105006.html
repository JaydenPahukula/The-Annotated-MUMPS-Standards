5.5 XMUMPS Specific Syntax

$&XMUMPS.Allocate(xdeclaration)

Provides memory allocation for the size of the X Window System data type specified by xdeclaration. The glvn will receive an xpointer from this function that represents a mechanism by which MUMPS routines may reference this data space.
