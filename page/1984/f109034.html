8.5.5 NEXTFOCUS einfoattribute

Definition During call back processing for an unfocus or change event, NEXTFOCUS identifies the window or gadget for which a related focus event will occur. If no related focus event will occur for a window or gadget that is defined in ^$WINDOW, NEXTFOCUS is equal to emptystring.
Domain focusloc
Access Reference
Default Not Applicable.
