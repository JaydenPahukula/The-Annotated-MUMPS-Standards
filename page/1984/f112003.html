11.1.3 MDC Strictly Conforming Implementation

An MDC strictly conforming implementation is an MDC conforming implementation that implements only MDC documents with MDC Type A status. An MDC strictly conforming implementation may not permit an application to modify the values of attributes for which this International Standard does not specify Modify access.
