postcond

See postcond for formal definition.

  1. Set semantic action flag Comact False.

Note: If argcond returns a condition (tvexpr is False), postcond inhibits execution of all semantic actions for the command until the arguments terminate. However, all syntactic paths are traversed as normal (that is, syntax checking is performed).
