5.4 XMOTIFRM Specific Syntax

The routines in this package begin with the letters Xrm or Mrm followed by a mixed case string of characters, the first of which is capitalized, representing the Motif Resource Manager routine to be executed. The routinenames used in this binding are those in the Motif software as provided by the Open Software Foundation. The exact list is not included herein. Those objects specified at this level that do not begin with the characters Xrm or Mrm are not included in this binding. Those entities in the Motif software that are defined as macros beginning with the characters Xrm or Mrm also are not included. Also included, related to items used at this level, are the associated data structures, which will be frequently accessed using the XMUMPS package.
