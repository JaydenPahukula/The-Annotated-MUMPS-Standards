$ASCII function

See $ASCII for formal definition.

  1. Result A.
  2. One ("1") B.
  3. Result B.
  4. If A is the null string, place –1 into Result.
    Otherwise, place the integer n associated with the character $E(A,B) into Result such that $A($C(n))=n.
