4.16.4 GQMNTN, Inquire maximum normalization transformation number

control-
mnemonic		   Level   GKS function name
GQMNTN 0a Inquire maximum normalization transformation number
No parameters
Write /GQMNTN
2 values returned
v1 ::= gksint error indicator
v2 ::= gksint maximum normalization transform number (1..n)
Read Error,MaxTran
