3.3.19 Focus Location

focusloc identifies a window or gadget which has, had, or is about to get focus.

    focusloc::= expr V │
│
│
│		 wname
wname , gname
emptystring		 │
│
│
│

If focusloc identifies a window, it has the form:

wname

If focusloc identifies a gadget, it has the form:

wname , gname

If focusloc does not identify a window or gadget defined in ^$WINDOW, it has the form:

emptystring
