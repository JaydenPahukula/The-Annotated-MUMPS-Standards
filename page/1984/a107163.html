7.1.6.5.38 $%MTXCOF^MATH

MTXCOF^MATH : REAL ( .A : MATRIX , I : INTEGER , K : INTEGER , N : INTEGER )

$%MTXCOF^MATH computes the cofactor in matrix A[N,N] for element A(I,K). The return value is the value of the cofactor.
