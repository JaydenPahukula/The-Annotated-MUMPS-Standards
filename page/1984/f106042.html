6.4.22 HROWID elattribute

Definition

In a table gadget, any row whose ritem does not follow the value of HROWID (in the collating sequence of ^$WINDOW) is a header row. Header rows do not scroll and are displayed in a manner visually distinct from non-header rows.

If HROWID is not defined, there are no header rows.

If an application attempts to modify HROWID to identify a row containing one or more choices that are currently selected, those choices are deselected.

Domain ritem
Access Create, Modify, Reference
Default None.
