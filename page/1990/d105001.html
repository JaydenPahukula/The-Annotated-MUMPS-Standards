5 Package Definition and Specific Syntax
5.1 XLIB Specific Syntax

All of the routines in this package begin with the letter X followed by a mixed case string of characters, the first of which is capitalized, representing the XLIB routine to be executed. The routinenames used in this binding are those provided in the X Window System as provided by the X Window Consortium. The exact list is not included herin. Those externrefs specified at this level that do not begin with an X are not included in this binding. Any other externref identified as a macro in the X Window Consortium documentation is not included. Routines whose routinenames that begin with Xrm are not included in this package, but are included in the XMOTIFRM package.
