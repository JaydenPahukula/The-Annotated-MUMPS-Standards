8. Data Exchange

Using the Data Exchange capabilities of MWAPI an M[UMPS] process can access items in a data source and can itself provide data sources. In either case a link must be established between the user and provider of a data source. The user of a data source, or client, initiates a link, while the provider, or server, responds to such initiation.
