5.4.23 Close device

(Operation type 42) relinquishes ownership of a device granted by a precedent open device transaction. The device parameters of the request pertain only to the mnemonic space contained in the request. The fields of the close device request and their sequence shall be:

  1. Environment: <LS> the server’s environment for the device.
  2. Device name: <LS> the server’s name for the device from the open device response.
  3. Client identifier: <SS> $Job (see ANSI/MDC X11.1) of the client process making the request, an ASCII string of decimal digits.
  4. Style: <SI> 0 denotes the positional style, and 1 denotes the keyword style of device parameters. All other values are reserved.
  5. Device parameters: <LS> as defined in 5.3.7.
  6. Mnemonic space: <SS> the server’s name for the mnemonic space to which these device parameters apply.
