4.1.12 SPECTRUM dattribute

Definition If the value of the display’s COLORTYPE dattribute is equal to "COLOR", SPECTRUM specifies the number of distinct color values (that is, the number of combinations of rcolor, gcolor and bcolor values that produce distinct colors) supported by the display. If the value of the display’s COLORTYPE dattribute is equal to "GREY", SPECTRUM specifies the number of distinct grey-scale values supported by the display. If the value of the display’s COLORTYPE dattribute is equal to "MONO", SPECTRUM is equal to 2.
