2.2.3 Numeric Literal numlit

The integer literal syntax, intlit, which is a nonempty string of digits, is defined here.

    intlit::= digit ...

See 2.1 for the definition of digit.

The numeric literal numlit is defined as follows.

    numlit::= mant [ exp ]
