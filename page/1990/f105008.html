5.4.2 ACTIVE wattribute

Definition

If ACTIVE is FALSE, the window and its descendant windows and elements are disabled. If a window or element is disabled, the user cannot interact with it and events do not occur for it.

If ACTIVE is TRUE for a window and all of its ancestor windows, if any, the window is not disabled.

Domain truefalse
Access Create, Modify, Reference
Default TRUE
