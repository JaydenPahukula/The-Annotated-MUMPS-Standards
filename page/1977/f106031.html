6.4.11 COLWIDTH elattribute

Definition

COLWIDTH specifies the default width for columns in a gadget.

If the unit of measure is CHAR, the basis font is the font specified by the gadget’s FFACE, FSIZE and FSTYLE elattributes. If the unit of measure is REL, the basis size is the size of the gadget at creation.
