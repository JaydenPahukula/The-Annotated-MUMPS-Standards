6.4.36 ROW elattribute

Definition

ROW specifies the rows in a table gadget, via descendant nodes of the form

^$W[INDOW] ( wname , gkeyword , gname , rowkeyword , ritem )

where

    rowkeyword::= expr V ROW

No value is present at the level of the ROW elattribute itself. There is a value of emptystring at the ritem level.

A ROW elattribute descendant is present for every ritem subscript that is present as a descendant of the CHOICE elattribute for the gadget. If an application assigns a CHOICE descendant with an ritem subscript that is not present as a ROW descendant, a ROW descendant with that ritem subscript is automatically created.

A ROW elattribute descendant may be present with a ritem subscript which is not present as a descendant of the CHOICE elattribute for the gadget. Such a ritem subscript defines an empty row.

Domain emptystring
Access Create, Modify, Reference
Default Not Applicable.
