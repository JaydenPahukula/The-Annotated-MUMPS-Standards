6.4.21 HCOLID elattribute

Definition

In a table gadget, any column whose citem does not follow the value of HCOLID (in the collating sequence of ^$WINDOW) is a header column. Header columns do not scroll and are displayed in a manner visually distinct from non-header columns.

If HCOLID is not defined, there are no header columns.

If an application attempts to modify HCOLID to identify a column containing one or more choices that are currently selected, those choices are deselected.

Domain citem
Access Create, Modify, Reference
Default None.
