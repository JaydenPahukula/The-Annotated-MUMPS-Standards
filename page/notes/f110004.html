9.1.3 $WTWIDTH

$WTWIDTH ( expr , fontface , fontsize , fontstyle , unitspec )

$WTWIDTH returns a value equal to the width of the text string expr displayed using the font indicated by fontface, fontsize and fontstyle. The width is expressed in the unit of measure indicated by unitspec.

If information is not available for the specified font, the return value is equal to the emptystring.

If unitspec is not equal to "PIXEL" or "POINT", an error condition occurs with $ECODE containing ",M?8,".
