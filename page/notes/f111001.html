10 Error Handling

Errors that occur during use of the MWAPI are handled in accordance with the error processing mechanisms specified for M[UMPS].

If one or more MWAPI errors occurs, at least one error is reported. If one or more errors occurs during the execution of an M[UMPS] MERGE command that references ^$DISPLAY or ^$WINDOW as a glvn on the left hand side of the equal sign, the resulting state of ^$DISPLAY or ^$WINDOW is not defined.
