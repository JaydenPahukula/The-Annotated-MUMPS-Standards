6.3.4.1.6 TIMER

The event class TIMER contains events that are generated when a specified interval has elapsed after a timer was set (see ^$Event). evid values are names. The implementor may limit the number of concurrent timers available, either by a single process or by the entire M[UMPS] system, or both.
