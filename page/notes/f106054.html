6.4.34 PTYPE elattribute

Definition

PTYPE specifies the pointer appearance when the pointer is in the gadget. A gadget’s PTYPE elattribute takes precedence over its parent window’s PTYPE wattribute.

If the display on which the gadget appears does not have a pointer, the value of PTYPE is ignored.

Domain ptype
Access Create, Modify, Reference
Default None
