4.21.3 GQPX, Inquire pixel

control-
mnemonic		   Level   GKS function name
GQPX 0a Inquire pixel
2 parameters
p1 ::= gksnam workstation ID
p2 ::= gkspnt pixel point (WC)
Write /GQPX("WORK1","100,100")
2 values returned
v1 ::= gksint error indicator
v2 ::= gksint colour index (–1..n)
Read Error,Color
