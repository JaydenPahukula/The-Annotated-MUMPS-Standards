1.2 X Window System and Motif

The X Window System architecture contains several layers. The lowest of these is the X protocol (under review in ANSI X3H3.6). The next two higher levels are the XLIB and XToolkit. These levels are included in FIPS 158. The next higher level is Motif, comprised of two packages (Motif and Motif Resource Manager) and has been adopted by the Open Software Foundation as a defacto standard for all platforms. IEEE 1295.1 is currently balloting on this. The software considered is that as available from the X Window Consortium and Open Software Foundation, both open membership organizations.
