4.1.9 FSTYLE dattribute

Definition FSTYLE indicates the default value for the FSTYLE wattribute for windows that are children of the display.
Domain fontstyle
Access Modify, Reference
Default Platform. The font style provided by the platform when requested for a FFACE of "M,DEFAULT", a FSIZE of 12 and a FSTYLE of "NORMAL".
