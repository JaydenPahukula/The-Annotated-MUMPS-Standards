6.4.4 CANCHANGE elattribute

Definition If CANCHANGE is TRUE, user actions can cause the VALUE attribute for the gadget to change. If CANCHANGE is FALSE, user actions cannot cause the VALUE attribute for the gadget to change, although some user interaction with the gadget may be possible.
Domain truefalse
Access Create, Modify, Reference
Default TRUE
