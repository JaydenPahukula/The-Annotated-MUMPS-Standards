9.5.5 GROUP einfoattribute

Definition GROUP allows related build-mode events to be recognized. A single user action may cause multiple selection and/or deselection events, or multiple move events. All events caused by a single action have the same value of GROUP. The actual value is arbitrary.
Domain intlit
Access Reference
Default Not Applicable.
