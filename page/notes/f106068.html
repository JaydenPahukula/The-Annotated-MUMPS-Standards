6.4.32 TFCOLOR elattribute

Definition TFCOLOR specifies the color for the gadget’s title text. If the underlying windowing platform does not enable the title text to have a different color from the remainder of the gadget’s text, the value of the TFCOLOR elattribute is ignored.
Domain color
Access Create, Modify, Reference
