gvnexpr

Introduced in the 1995 ANSI M[UMPS] language standard.

This metalanguage symbol represents an expression that evaluates to the name of a global variable.

Approved for addition in a future M[UMPS] language standard.

The name of the global variable, in this context, is unsubscripted.
