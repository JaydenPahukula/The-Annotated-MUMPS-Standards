5.4.16 MENUBAR wattribute

Definition MENUBAR identifies the menu to be displayed in the window’s menu bar. If the MENUBAR attribute is not specified when a window is created, no menu bar is created for the window. If the underlying windowing platform prevents a menu bar from being added after window creation, attempts to modify the MENUBAR attribute are ignored. If the value specified for MENUBAR is not the mname of a menu which is a child of the window, or if the value is equal to emptystring, an empty menu bar is created.
Domain mname or emptystring
Access Create, Modify, Reference
Default None
